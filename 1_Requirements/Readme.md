8x8- LED Martrix
================

## Introduction
------------
A simple 8x8 LED Matrix interfacing to display the English characters using the AVR toolchain without the Arduino IDE.

## Software
 * SimulIDE
 * Visual studio code
 
## Installation
------------
You will need to have the AVR toolchain installed on your system. This includes:
* avr-gcc
* avr-libc
* avrdude
